After we receive these inputs we would make several API calls to google maps, yelp, and open weather and return the results to the user.

Call of Duty Warzone Stats

This is a personal project that I started just for fun. I really enjoy playing this game with friends especially during COVID and am very data focused in general. So, when I found out I could make an API call and get all of the in game stats I wanted to build a little site around it. On this site you can enter your gamer tag and platform you play on and it will return your current key stats as well as your last 20 game stats.
